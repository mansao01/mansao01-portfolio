Triangle Sneacare API

The Triangle Sneacare API is a backend service that supports the Triangle Sneacare application, designed to manage transactions and other related operations. This API is built using the RESTful architecture with technologies like Node.js, Express, and MySQL. It integrates with the Midtrans Core API for secure payment processing and utilizes JSON Web Tokens (JWT) for robust authentication and authorization, ensuring secure access to protected resources.

JavaScript restful-api MySQL Express Node.js midtrans-core-api
